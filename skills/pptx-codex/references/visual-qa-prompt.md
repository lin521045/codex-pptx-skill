# 视觉 QA 提示模板

将 slide 渲染为单页图片后，使用下面这段检查提示。

```text
请作为第二轮视觉审查者，逐页检查这些幻灯片图片，默认认为一定存在问题。

重点寻找：
- 元素重叠
- 文本溢出或被裁切
- 标题换行后与装饰元素冲突
- 页脚、来源或注释与正文相撞
- 内容块之间距离过近（小于约 0.3"）
- 左右不对齐或上下节奏不均衡
- 页面边缘安全距离不足（小于约 0.5"）
- 低对比文字或图标
- 图表、图片、卡片数量与正文不匹配
- 残留 placeholder 文本

请按页输出：
1. 发现的问题
2. 严重程度
3. 推荐修复方向

即使是轻微问题也要指出，不要默认“看起来差不多就算通过”。
```
