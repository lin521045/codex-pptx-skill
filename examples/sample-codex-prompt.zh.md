# 示例 Codex Prompt

下面是一段可以直接复制给 Codex 的中文示例 prompt。

```text
请使用 $pptx-codex 帮我制作一份 8 页左右的汇报 PPT。

主题：
AI 辅助科研写作工作流

受众：
高校教师与研究生

目标：
说明为什么需要这套工作流、它解决了什么问题、最终能带来哪些收益。

风格要求：
- 专业
- 现代
- 不要通用 AI 模板感
- 尽量用卡片、时间线、对比和大数字，不要整页堆 bullet

请按以下方式完成：
1. 先给我页结构建议。
2. 再给出配色和字体方向。
3. 如果没有模板，就用 PptxGenJS 从零创建一个 starter deck。
4. 生成后做内容 QA 和视觉 QA。
5. 最后说明哪些地方还值得我补充素材。
```
